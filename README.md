# aoc-2025

To install dependencies:

```bash
bun install
```

To create a new day file:

```bash
bun run scripts/newDay.ts XX
```

To run day XX part 1 with full input data:

```bash
bun run src/dayXX.ts 1 full
```

Quick copy-paste to run a day - fill with day number(padded).ts, part(1 or 2), and input type('full' to do full data run, otherwise test):
```bash
bun run src/day
```
This project was created using `bun init` in bun v1.3.3. [Bun](https://bun.com) is a fast all-in-one JavaScript runtime.
